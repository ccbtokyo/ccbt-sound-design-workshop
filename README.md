# ひらめく☆サンプラー  
CCBTで行われているワークショップ「**ひらめく☆道場　サウンドデザイン入門**」に関連し、開発されたシステム。  
「ひらめく☆サンプラー」を使うと、パソコンやスマートフォンなどの画面上で録音、再生、ループなどができます。
